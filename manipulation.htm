Basic Image Manipulation

This first assignment was an introduction to Photoshop, the workspace and some of the basic tools it offers. A solid understanding of these basics is crucial to creating any successful Photoshop project.

This exercise has taught me the importance of performing adjustments on images to prepare them for the web. Proper alignment, cropping and image size require only a little bit of work but in the big picture can make all the difference. The internet and its users have little patience, and even a few extra seconds of download time can cause them to look elsewhere. So when the size of the image can make or break a webpage, you really need to ensure that the files are fully optimized. The images below are a good example of the huge impact that can be made. Finding that sweet spot balance between file size and image quality may change depending on the website, but what doesn't change is that size does matter!

Image 8a

  • Image 8a was optimized using the Photoshop "Save As..." function.
  • Dimensions: 215px x 162px.
